PROJETO PODCAST GERADO POR IA's
------------------------------------------------------------

Título: Universia - Fundamentos de IA Generativa em parceria com a DIO.
------------------------------------------------------------

Projeto: Podcast Expedição 404 

Observação:
Este repositório foi desenvolvido como um fluxo de trabalho para criação de podcast utilizando ferramentas de inteligência artificial em todas as etapas: roteiro, voz, artes e edição. Ele serve como base para estudo ou reprodução da mesma esteira criativa.

Objetivo do projeto:
Criar um podcast completo utilizando recursos de IA para otimizar todo o processo criativo, abordando temas como tecnologia, inteligência artificial, design, acessibilidade e experiência digital.
------------------------------------------------------------

Tecnologias utilizadas no projeto:

ChatGPT (prompts , roteiro)
MidJourney (imagens)
ElevenLabs (áudio, narrativa)
AI Music Generator (música)
CapCut (edição final de áudio)
------------------------------------------------------------

Como o projeto foi desenvolvido:

O roteiro foi gerado usando o ChatGPT.

A narração foi produzida na ElevenLabs utilizando o texto final e a música foi produzida com o AI Music Generator.

As capas e o avatar foram criados com MidJourney seguindo prompts específicos.

A edição de áudio foi feita no CapCut para adicionar trilhas, cortes e efeitos.
